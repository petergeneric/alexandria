# History Firefox Extension

Firefox extension for capturing browsing history page content.

**Status**: Not yet implemented (Phase 4).

## Planned Functionality

- Capture page text content from the DOM on navigation
- Send snapshots to the Rust backend via native messaging
- Extension popup for quick search
